## Introduction

Manages a Windows Web App.

## File Structure 

This particular module is having the following configuration files.
- webapp_windows.tf : This file is having the terraform code to provision the resource as per requirements.
- variables.tf : This file consists of all the variable definitions needed for the terraform code mentioned in webapp_windows.tf
- outputs.tf : This file will be having the resource output details generated.

## Requirements

1. Resource group
2. App service plan
